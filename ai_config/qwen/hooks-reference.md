## TOP-20 Hooks (приоритизированный список)

| # | Hook | Когда срабатывает | Зачем |
|---|------|-------------------|-------|
| 1 | **PreToolUse** | Перед выполнением инструмента | Блокировка опасных действий, валидация |
| 2 | **PostToolUse** | После выполнения инструмента | Автоформатирование, линтинг |
| 3 | **Stop** | Завершение ответа Claude | Запуск тестов, генерация summary |
| 4 | **SubagentStop** | Завершение субагента | Валидация вывода субагента |
| 5 | **Notification** | Отправка уведомления | Алерты в Slack/email |
| 6 | **Pre-commit** | Перед git commit | Тесты, линтер, форматирование |
| 7 | **Post-commit** | После git commit | Уведомление, сборка |
| 8 | **Pre-push** | Перед git push | Полная проверка, security scan |
| 9 | **Post-merge** | После git merge | Миграции, сборка, уведомления |
| 10 | **FileChange** | Изменение файла | Авто-формат, обновление связанных файлов |
| 11 | **DirectoryCreate** | Создание директории | Проверка структуры, инициализация |
| 12 | **ErrorDetected** | Обнаружена ошибка | Авто-поиск решения, алерт |
| 13 | **TestFailure** | Тест упал | Авто-диагностика, предложение фикса |
| 14 | **BuildFailure** | Сборка упала | Анализ логов, предложение решения |
| 15 | **SecurityViolation** | Нарушение безопасности | Блокировка, алерт команде |
| 16 | **PerformanceDegradation** | Ухудшение performance | Профилирование, поиск bottleneck |
| 17 | **DependencyUpdate** | Обновление зависимости | Проверка breaking changes, тесты |
| 18 | **ContextSwitch** | Смена задачи | Сохранение контекста, подготовка нового |
| 19 | **SessionStart** | Начало сессии | Загрузка контекста проекта, memory |
| 20 | **SessionEnd** | Конец сессии | Сохранение memory, генерация summary |

## Конфигурация хуков

Хуки настраиваются в `.claude/settings.json` и поддерживают три типа обработчиков:

1. **command** — shell-скрипты (exit code 0 = approve, non-zero = deny)
2. **prompt** — одношаговая LLM-оценка контекста через `$ARGUMENTS`
3. **agent** — глубокий анализ кодовой базы через выделенный субагент

## Пример конфигурации

```json
{
  "hooks": {
    "PreToolUse": {
      "command": "node hooks/pre-tool-use.js"
    },
    "PostToolUse": {
      "command": "npx prettier --write $FILEPATH"
    },
    "Stop": {
      "command": "npm test"
    },
    "SubagentStop": {
      "agent": "validate-subagent-output"
    }
  }
}
```
